## Running:

- WASD to move around
- Use the mouse to look around
- ESC to exit

## Libraries in use:

- GLFW3 & GLAD for OpenGL and rendering.
- GLM for matrix calculations.
- Freetype2 for font rendering.
- Convex Hull 3D library (https://github.com/leomccormack/convhull_3d)
